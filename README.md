# vajer

Tokenize lisp.

## Installation

```
git clone --recursive https://github.com/saikyun/vajer
cd libtcc-example
cd lib/tinycc
./configure --config-codesign
make
make test
cd ..
```

## Usage

```
make test
```

On windows, you'll have to set up the compiler flags yourself. If you figure it out, please send a PR. :)
